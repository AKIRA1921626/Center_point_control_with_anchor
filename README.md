# 中心点アンカー移動.anm2と中心点アンカー移動_回転付き.anm2

オブジェクトの **中心点をアンカーを使って操作** できるようにしたスクリプト。  
中心点アンカー移動.anm2は中心点の移動を行うだけ。  
中心点アンカー移動_回転付き.anm2は中心点の移動を行う機能と、回転も操作できる機能がセットになっている。  
前者は比較的機能が少なく、後者は機能が多いだけで性能に差はない。  
このスクリプトで中心点を移動させるとき、中心点だけが移動し、オブジェクトの見かけ上の位置は移動しないようにした。  
チェックボックスを使用することで、見かけ上の位置を移動させるようにすることも出来る。  
アンカーだけではなく、トラックバーによる操作、上下左右9箇所への固定も可能。  
 <u>「移動距離をパーセンテージにする」</u> のチェックボックスがONの場合は、x,yの移動する距離をpxではなく、元のオブジェクトの％で行う用になる。  
	中心Xが50ならばx方向に元の画像の50%分すすむ。  
トラックバーの数値は<u>「トラックバーの数値を無効化」</u>のチェックボックスで無効にできる。  
<u>「位置補正」</u>のチェックボックスがOFFならば、中心点は移動せず、オブジェクトだけが移動する。  
中心タイプを<mark>「回転重視」</mark>とすれば、設定した中心点を基準に回転を行うが、拡大は行わない。  
	「回転重視」の中心点で拡大を行いたい場合は、 **トラックバーの"中心Z"** で行う。  
中心タイプを<mark>「拡大重視」</mark>とすれば、設定した中心点を基準に拡大を行うが、回転は行わない。  
	また、位置が大きくずれるという問題もある。  
 **デフォルトでは「回転重視」** となっている。  
--label:配置としているので、配置の個所にスクリプトが追加されます。  
中身は200行ぐらいになっているが、AIにリファクタリング（コードを簡潔にする事）をしてもらえば５０行程度に納まるらしいので、改変などを行う場合などはそちらもご一考ください。  

二次配布可、改変可、クレジット不要。
