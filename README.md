# 中心点アンカー操作

オブジェクトの中心点をアンカーを使って操作できるようにしたスクリプト。  
その時、中心点だけが移動し、オブジェクトの見かけ上の位置は移動しないようにした。  
アンカーだけではなく、トラックバーによる操作、上下左右9箇所への固定も可能。  
ただし、拡大率を100以上にすると、オブジェクトの見かけ上の位置も移動してしまう。    
中心タイプを「回転重視」とすれば、設定した中心点を基準に回転を行うが、拡大は行わない。  
中心タイプを「拡大重視」とすれば、設定した中心点を基準に拡大を行うが、回転は行わない。  
デフォルトでは、「回転重視」となっている。  
--label:配置としているので、配置の個所にスクリプトが追加されます。  
二次配布可、改変可、クレジット不要。
